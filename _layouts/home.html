GPyOpt

-- Python open-source library for Bayesian Optimization --

-- Developed by the Machine Learning group of the Univeristy of Sheffield --

-- Based on GPy , python framework for Gaussian process modeling--

Why?

With GPyOpt you can:

  • Solve global optimization problems using Gaussian processes as surrogate models.

  • Import all GPy functionalities in your optimization. Choose between a large variety of kernels, kernel combinations, likelihoods, etc.

  • Run Parallel optimization methods and make use of all your computational resources and design batches of wetlab experiments.

  • Run Big Data optimization and use sparse Gaussian process and use Bayesian optimization with massive datasets in complex problems.

    • Installation

    The simplest way to install GPyOpt is using pip. Be sure that all the dependencies (GPy, numpy, scipy, matplotlib, pylab, random) are also installed.

    Ubuntu users can do: 
    >> sudo apt-get install python-pip
>> pip install GPyOpt

    GPyOpt is available for Mac users. Write (if you use homebew): 
    >> sudo brew install python-pip
>> pip install GPyOpt

    On Windows, we recommend the anaconda python distribution. From the anaconda command prompt type: 
    >> pip install GPyOpt

    Install from source and contribute: You can install GPyOpt from source. To fork the repository click fork on the GPyOpt project page. To contribute to the repository, clone it and add it to your pythonpath and send a pull request 
    >> git clone git@github.com:SheffieldML/GPy.git ~/SheffieldML
echo 'PYTHONPATH=$PYTHONPATH:~/SheffieldML' >> ~/.bashrc
